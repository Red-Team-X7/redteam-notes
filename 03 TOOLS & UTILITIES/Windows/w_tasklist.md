## Description

This tool displays a list of currently running processes on either a local or remote machine.

## Usage Examples

### Find task

	tasklist | findstr SERVER.EXE

```
SERVER.EXE	2196	0	3,196 K
SERVER.EXE	6056	0	3,192 K
```
